🚀 FIRST TIME SETUP (REQUIRES NETWORK):

Option 1: Let the setup wizard handle everything
────────────────────────────────────────────────
When you have network access, run:

    go mod download
    go build -o resignipa main.go
    ./resignipa setup

The setup wizard will:
• ✓ Verify system prerequisites
• ✓ Check all required tools
• ✓ Download dependencies
• ✓ Build the project
• ✓ Discover certificates
• ✓ Show next steps

Option 2: Manual setup
────────────────────────────────────────────────
If you prefer manual control:

    go mod download      # Download dependencies
    go mod tidy          # Organize dependencies
    make build           # Build the project

📦 WHAT'S NEEDED:

1. Internet Connection
   • To download Go dependencies (fyne, cobra, etc.)
   • Only needed once

2. System Requirements
   • macOS
   • Go 1.21+
   • Xcode Command Line Tools
   • Valid Apple Developer certificate (for actual resigning)

💡 WHY THIS ERROR?

You're seeing build errors because go.sum is missing. This file contains
cryptographic checksums of dependencies and is generated when you run:
• go mod download
• go mod tidy
• go get

This is a one-time setup requirement.

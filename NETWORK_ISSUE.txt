🔍 TROUBLESHOOTING NETWORK ISSUES:

If you see "socket is not connected" errors:

1. Check your internet connection
2. Try again in a few minutes (network might be unstable)
3. Use a different network if available
4. Check if you're behind a proxy (set GOPROXY if needed)

Set Go proxy (if needed):
────────────────────────
export GOPROXY=https://proxy.golang.org,direct
go mod download

Or use direct mode:
──────────────────
export GOPROXY=direct
go mod download
